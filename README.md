# Products API

This is a simple **Laravel-based API** for managing product data. This project includes basic CRUD operations and is built with PHP and Laravel framework.

## Requirements

- PHP 8.x or higher
- Composer
- MySQL / MariaDB
- Laravel 9.x or higher
- Git
